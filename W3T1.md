## ğŸ’¡ Approach
- Store students in a `List`
- Use Java **Stream API**
- Sort using:
  - `Comparator.comparingInt().reversed()` for marks
  - `thenComparing()` for name
- Use `limit(K)` to get top students
- Map to names and collect results

## ğŸ’» Java Implementation
```java
import java.util.*;
import java.util.stream.*;

class Student {
    String name;
    int marks;

    Student(String name, int marks) {
        this.name = name;
        this.marks = marks;
    }
}

public class TopKStudents {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        List<Student> students = new ArrayList<>();

        for (int i = 0; i < n; i++) {
            String name = sc.next();
            int marks = sc.nextInt();
            students.add(new Student(name, marks));
        }

        int k = sc.nextInt();

        List<String> result = students.stream()
                .sorted(
                        Comparator.comparingInt((Student s) -> s.marks).reversed()
                                .thenComparing(s -> s.name)
                )
                .limit(k)
                .map(s -> s.name)
                .collect(Collectors.toList());

        System.out.println(String.join(" ", result));
        
    }
}
```
